音乐解锁

Created2025-04-24|Updated2025-05-15|Tutorials
|Word Count:1.4k|Reading Time:4mins|Post Views:

建议使用非中国大陆ip进行访问。

目前支持 网易云音乐(ncm), QQ音乐(qmc, mflac, mgg), 酷狗音乐(kgm), 虾米音乐(xm), 酷我音乐(.kwm)

Demo: https://demo.unlock-music.dev/

React Demo: https://um-react.netlify.app/

代码仓库: https://git.unlock-music.dev/um/

Cli版(无GUI界面，本地批量处理): https://git.unlock-music.dev/um/cli

QQ音乐

重复下载同一首的歌曲不重复扣下载配额，但是同一首歌的两个版本会重复扣下载配额，请仔细分辨。

部分平台获取的加密文件未包含密钥。选择你下载文件时使用的客户端来查看说明。

Windows

目前 Windows 客户端 19.51 或更低版本下载的歌曲文件无需密钥，其余平台的官方正式版本均需要提取密钥。

QQ音乐1943.exe安装包

QQ音乐屏蔽更新，桌面建个记事本，记事本重命名 “QQMusicUp.exe”

（其实就是用空文件替换掉原文件）

Mac

Mac 需要降级到 8.8.0 或以下版本。

导入密钥可以参考下面的步骤：

点击顶部的设置

设定区域选择QMCv2 密钥

按下 添加一条密钥 按钮 右侧 的 v

选择 从文件导入密钥…

查看密钥提取说明：

Mac 客户端使用 mmkv 数据库储存密钥。

此外，你需要降级到 2023.09.03 版本的客户端。新版本对 mmkv 数据库进行了加密处理。

该密钥文件通常存储在下述路径：

~/Library/Containers/com.tencent.QQMusicMac/Data/Library/Application Support/QQMusicMac/mmkv/MMKVStreamEncryptId

导入密钥
选中并复制上述的 MMKVStreamEncryptId 文件路径

点击上方的「文件选择区域」，打开「文件选择框」

按下「shift + command + G」组合键打开「路径输入框」

粘贴之前复制的 MMKVStreamEncryptId 文件路径

按下「回车键」确认。

安卓（Android）

一般做法

安卓提取密钥需要 root 特权，建议用电脑操作。

QQ 音乐官方版本需要提取密钥才能解密。

你也可以尝试使用【QQ 音乐简洁版】或 OEM 定制版（如小米、魅族定制版）。简洁、定制版本目前不需要提取密钥。

导入密钥可以参考下面的步骤：

点击顶部的设置

设定区域选择QMCv2 密钥

按下 添加一条密钥 按钮 右侧 的 v

选择 从文件导入密钥…

查看密钥提取说明：

启动具有 root 特权的文件浏览器

访问 /data/data/com.tencent.qqmusic/databases/ 目录。

将文件 player_process_db 复制到浏览器可访问的目录。
（例如下载目录）

提交该数据库文件。

vivo手机做法

提供一种在vivo手机上提出qq音乐密钥的方法：

vivo手机可以用电脑版互传（网页版不行）单独把qq音乐(连带数据)备份出来。

记住备份文件的位置方便找到

然后用压缩软件打开。里面就有这个文件。

网易云音乐

除了使用音乐解锁之外，网易云音乐还可以使用无损解析的网站。

网易云音乐无损解析

telegram网易云音乐下载机器人

使用方式：

1.输入歌曲名称或歌曲ID或分享链接，点击搜索或直接解析

2.选择音质，点击解析

3.等待解析完毕，点击下载歌曲

酷我音乐

日前，仅手机客户端下载的至臻全景声及至臻母带音质的音乐文件采用新版加密。

其他音质目前不需要提取密钥。

PC平台暂未推出使用新版加密的音质。

安卓（Android）

启动具有 root 特权的文件浏览器

访问 /data/data/cn.kuwo.player/files/mmkv/ 目录。

将文件 cn.kuwo.player.mmkv.defaultconfig 复制到浏览器可访问的目录。
（例如下载目录）

提交该数据库文件。

酷狗音乐

酷狗现在对部分用户推送了 kgg 加密格式（安卓、Windows 客户端）。

根据平台不同，你需要提取密钥数据库。

安卓（Android）

启动具有 root 特权的文件浏览器

访问 /data/data/com.kugou.android/files/mmkv/ 目录。

将文件 mggkey_multi_process 复制到浏览器可访问的目录。
（例如下载目录）

提交该数据库文件。

Windows

酷狗的 Windows 客户端使用 SQLite 数据库储存密钥。

该密钥文件通常存储在下述路径：

%APPDATA%\KuGou8\KGMusicV3.db

Cli版不会用？

  1. 根据使用的平台下载好对应的压缩包并解压
  2. 把需要解密的歌曲拖拽进um.exe
  3. 解密后的歌曲会保存到原目录下

（也就是你存放需要解压的歌曲的地方，不是um.exe所在的文件夹）

（PS：批量处理没反应那就一首一首来）

下载限额用完了

小妙招解决大问题，就是有（亿点点麻烦）如下：
QQ音乐缓存试听路径（下载限额用完，用这个）：
Android/data/com.tencent.qqmusic/cache/DataTransport/2810304/data/tp2p/.tpfs/duty/F0M0003fqwIn2Gqrci.mflac

密钥路径在root权限 试听文件名必须与密钥对应：
/data/data/com.tencent.qqmusic/databases/player_process_db

小妙招解决大问题（没难度）如下：
酷狗音乐试听路径下载限额用完（用这个）
Android/data/com.kugou.android/filse/kugou/down_c/default/0bb56882ffab74d88c86741b4a0ad8d2-2-630243241_HQ_force.kge（选择什么音质就缓存什么音质。）可能播放会跳转时间，但是不影响听。
