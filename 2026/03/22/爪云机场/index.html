用ClawCloud Run搭建自己的免费机场

发表于2026-03-22|更新于2026-03-22|Tutorials
|总字数:593|阅读时长:1分钟

视频教程：https://www.youtube.com/watch?v=4BuQRBmyYJs&t=569s

前置补充

clash类应用不一定支持这种订阅链接，因此在代理软件的选择上：

  1. Android端推荐使用nekobox

  2. Windows、Mac、Linux推荐使用v2RayN

  3. 如果嫌麻烦可以全部用Karing（全平台通用）

第一步，注册ClawCloud Run账号

注意，最好要有一个注册满180天的GitHub账号（不然没有每月免费的5美元额度）

如何查看GitHub注册天数？

在浏览器地址栏输入：https://api.github.com/users/你的用户名

找到返回 JSON 中的 created_at 字段，格式为 YYYY-MM-DDTHH:MM:SSZ（UTC 时间）

用这个日期与今天日期计算天数差即可

注册地址：https://run.claw.cloud/

在页面左上角选择服务器的地址。（这里建议选US WEST，离国内近，而且人少一些，新加坡服务器人太多了）

第二步，部署项目

项目一（vless）

  1. 点击App Launchpad，点击右上角的Create App。
  2. Application Name随便改，这是项目的名字。
  3. Image Name改成7techlife/seven。
  4. Usage把CPU和内存都拉到最小。（牺牲点稳定性优先保障流量）
  5. 其他不动，点击Deploy Application
  6. 部署完成后进入项目，在Overview最下面点击Logs。（要等Status是Active再弄）
  7. 在Logs界面可以选择复制单个节点链接导入代理软件，也可以复制聚合链接文本导入全部。

项目二（Hysteria2）

  1. 点击App Launchpad，点击右上角的Create App。
  2. Application Name随便改，这是项目的名字。
  3. Image Name改成7techlife/sevenhy2。
  4. Usage把CPU和内存都拉到最小。（牺牲点稳定性优先保障流量）
  5. Network的Container Port填写7777，开启Public Access，把后面链接的https前缀改选成udp
  6. Advanced Configuration的Environment Variables添加password=自定义密码，防止应用重启导致订阅链接更改。
  7. 其他不动，点击Deploy Application
  8. 部署完成后进入项目，在Overview最下面点击Logs。（要等Status是Active再弄）
  9. 在Logs界面复制提供的链接模板，导入代理软件。
  10. 进入项目，在Overview最下面找到Network-Public Address（右边那个），复制下面的udp链接
  11. 在代理软件里把这条订阅链接的地址改成成udp链接（并删掉udp://和原链接后面的：端口号），再把端口号设置成udp链接里的端口（不是7777的那个），点击保存即可。
